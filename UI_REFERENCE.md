## Visual Design System

### Text Styles

**Outlined Label** (Custom):
```swift
// White text with black stroke
struct OutlinedText: View {
    let text: String

    var body: some View {
        ZStack {
            // Stroke (outline)
            Text(text)
                .foregroundColor(.black)
                .overlay(
                    Text(text)
                        .foregroundColor(.black)
                        .offset(x: 1, y: 1)
                )

            // Fill (white)
            Text(text)
                .foregroundColor(.white)
        }
    }
}
```

**Typography Scale**:
- Large Title: 34pt, Bold
- Title: 24pt, Semibold
- Headline: 17pt, Semibold
- Body: 17pt, Regular
- Caption: 12pt, Regular
- Tiny: 10pt, Regular

### Spacing System
- XXS: 4pt
- XS: 8pt
- S: 12pt
- M: 16pt
- L: 24pt
- XL: 32pt
- XXL: 48pt

### Corner Radius
- Small: 8pt (buttons, cards)
- Medium: 12pt (modals)
- Large: 16pt (panels)

### Shadow Styles
```swift
// Card shadow
.shadow(color: .black.opacity(0.1), radius: 8, x: 0, y: 2)

// Modal shadow
.shadow(color: .black.opacity(0.2), radius: 16, x: 0, y: 4)

// Button shadow
.shadow(color: .black.opacity(0.15), radius: 4, x: 0, y: 2)
```
